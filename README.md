# puyue-LSTM
用 python 写一个预测股价的算法
使用一种叫做 长短期记忆网络（LSTM） 的人工神经网络来处理时间序列数据，如股票价格。
LSTM 可以记住过去的信息，并根据当前的输入来预测未来的输出

其中使用了 tensorflow 这个模块 必须要x86设备才能运行(mac不行)
python3.8(3.9和以后不能用)

usage
pip install -r requirements.txt 
python main.py
